About This Brain Tumor Detection Project

This web application utilizes a deep learning model to analyze MRI images and predict the presence of different types of brain tumors. The model is built using the Xception architecture, trained on a dataset of MRI images labeled with four classes: glioma tumor, meningioma tumor, pituitary tumor, and no tumor.

By uploading an MRI image, users can get a prediction of whether a tumor is present and the model's confidence level in its prediction. The application provides a simple interface to interact with the model and displays the results in an easy-to-understand format.

This project was created to demonstrate the use of convolutional neural networks (CNNs) in medical image classification, specifically for detecting brain tumors. It aims to provide a glimpse into how artificial intelligence can assist in the field of medical diagnostics.

Algorithm and Model Training:

The model uses a custom-built architecture based on the Xception model, which is known for its effectiveness in image classification tasks. The training process involved the following steps:

  1. Data Preparation: MRI images were preprocessed, resized to 150x150 pixels, and organized into training and testing datasets.
  2. Model Construction: The Xception model was utilized as a feature extractor, followed by global average pooling and fully connected layers for classification.
  3. Training: The model was trained for 12 epochs, achieving an accuracy of over 98% on validation data.
  4. Evaluation: The performance was evaluated using metrics such as accuracy, loss, and a confusion matrix.
